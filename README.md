## 🎯 项目简介

**妆策AI（ZhuangCe AI）** 是一个面向美妆新零售实战场景的 **AI驱动智能推荐与营销转化平台**，为大学生创业竞赛（美妆新零售实战赛）打造。

项目以 **DEAR SEED 玫瑰修护洗发水** 为MVP示例产品，聚焦抖音电商赛道，面向 **18-22岁学生党女性** 群体，通过 **15+ AI算法** 实现从选品评估、智能推荐、内容生成到运营优化的全链路数据驱动决策。

### 🔥 解决的核心痛点

| 痛点 | 传统方式 | 妆策AI方案 |
|:---:|:---:|:---:|
| 🎲 选品靠感觉 | 经验判断，试错成本高 | **六维算法评分** · GBDT AUC=0.9993 |
| 📝 内容同质化 | 人工撰写，爆款难复制 | **UCB/Thompson最优臂** · 转化率3.67% |
| ⏰ 时机全靠猜 | 随机投放，ROI低 | **ARIMA时序预测** · 开学季boost=1.15 |

## 🚀 快速开始 / Quick Start

### 环境要求

- Python 3.10+
- Node.js 18+ (前端)
- pip

### 安装 & 运行

```bash
# 1. 克隆仓库
git clone https://github.com/sjkncs/zhuangce-ai.git
cd zhuangce-ai

# 2. 安装后端依赖
cd zhuangce-ai-project/backend
pip install -r requirements.txt

# 3. 启动后端
python app.py

# 4. 安装前端依赖（新终端）
cd ../frontend
npm install

# 5. 启动前端
npm run dev
```

### 运行数据分析脚本

```bash
cd zhuangce-ai-project/data_analysis

# 按顺序运行（01是前置依赖）
python 01_data_cleaning.py
python 02_eda_analysis.py
python 03_predict_model.py
# ... 依次运行到 10_paper_algorithms.py

# 结果输出到 output/ 目录
```

### 生成竞赛PPT

```bash
# 在项目根目录
pip install python-pptx
python make_ppt.py
# 输出: Word版/妆策AI-竞赛答辩PPT-升级版.pptx (含演讲备注)
```
