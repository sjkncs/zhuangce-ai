# 妆策AI：面向美妆新零售实战场景的智能推荐与营销转化平台

## 项目定位
妆策AI是一个面向美妆品牌新品推广场景的智能增长平台。
项目聚焦“预测→决策→转化→复盘”这一最小闭环，帮助品牌回答四个关键问题：
- 什么值得推？
- 推给谁？
- 什么时候推？
- 推完之后如何复盘？

## 为什么做这个项目？
很多美妆品牌并不缺产品，缺的是增长判断力。
尤其在新品推广中，常见问题并不是“没有内容”，而是：
- 缺少对传播潜力的前置判断
- 缺少对高潜人群的识别
- 缺少从内容到转化的决策支持
- 缺少推广后的复盘逻辑

妆策AI尝试用公开内容样本、预测逻辑和推荐策略，帮助品牌用“预测”代替“试错”。

## 校赛 MVP 聚焦什么？
校赛阶段，我们不追求大而全，而是聚焦一个最小可行闭环：

**输入产品卖点 + 价格区间 + 目标人群 → 输出传播潜力评分 + 高潜人群画像 + 推荐商品 + 内容建议 + 复盘逻辑**

本轮 MVP 重点案例：
- **品牌场景：** 核心 DEAR SEED
- **重点品类：** 洗护
- **重点产品：** DEAR SEED 玫瑰修护洗发水（示例产品对象）

## 核心功能
### 1. 用户洞察
提取公开内容样本中的热词、用户标签、场景标签与价格偏好。

### 2. 爆款预测
基于样本数据和标签逻辑，对产品卖点的传播潜力进行判断。

### 3. 推荐决策
结合人群、场景与产品卖点，输出推荐商品、推荐场景与推荐平台优先级。

### 4. 内容种草
根据预测和推荐结果，生成小红书标题、社群话术与短视频开场文案。

### 5. 转化复盘
以可视化方式展示预测—推荐—内容—复盘链路，辅助答辩与策略说明。

## 样板案例（模拟验证）
### 输入
- 产品：DEAR SEED 玫瑰修护洗发水
- 卖点：修护 + 柔顺 + 玫瑰香氛
- 价格：100-150 元
- 人群：学生党
- 平台：小红书

### 输出
- 传播潜力评分：8.2 / 10
- 高潜人群：学生党女性用户
- 推荐场景：宿舍日常 / 开学季
- 推荐平台：小红书 > 社群 > 视频号
- 推荐商品：玫瑰护手霜（示例）

> 注：以上结果基于公开内容样本、标签体系与模拟验证逻辑生成，不等同于真实运营数据。

## 技术路线
- 数据采集：MediaCrawler
- 数据处理：Python / CSV / 标签标注
- 预测辅助：Qwen / LangChain（轻量推理与解释）
- 推荐逻辑：协同过滤思路 + 场景匹配 + 标签联动
- 展示层：Vue + ECharts

## 分类策略
为避免不同品类用户逻辑混杂，项目采用“一级品类—二级子类—标签层”的分类框架。

当前校赛阶段重点聚焦：
- 一级品类：洗护
- 二级子类：修护洗发
- 关键标签：修护、柔顺、学生党、宿舍、开学季、高性价比

## 当前说明
当前项目以校赛 MVP 为目标，重点完成概念验证与模拟验证。
因此：
- 部分结果来自公开样本数据
- 部分案例属于模拟推演
- 暂未替代真实品牌完整投放结果

后续如接入企业真实运营数据，可进一步做更高强度验证。
