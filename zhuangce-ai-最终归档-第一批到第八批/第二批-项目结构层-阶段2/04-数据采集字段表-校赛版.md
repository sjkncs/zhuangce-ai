# 妆策AI｜数据采集字段表（校赛版）

## 一、采集目标
围绕 DEAR SEED 洗护类产品相关关键词，采集小红书公开内容样本，用于热词分析、标签标注与模拟验证。

## 二、基础字段（必须）
| 字段名 | 含义 | 是否必须 | 备注 |
|---|---|---|---|
| id | 内容唯一标识 | 是 | 平台原始 ID 或自定义编号 |
| platform | 来源平台 | 是 | 小红书/抖音 |
| title | 标题 | 是 | 原始标题 |
| content | 正文/简介 | 是 | 原始文本 |
| publish_time | 发布时间 | 是 | 年月日即可 |
| like_count | 点赞数 | 是 | 用于热度评分 |
| comment_count | 评论数 | 是 | 用于热度评分 |
| collect_count | 收藏数 | 建议 | 小红书优先 |
| keyword | 检索关键词 | 是 | 便于回溯 |
| url | 原始链接 | 建议 | 若可保存则保留 |

## 三、结构化字段（人工补充）
| 字段名 | 含义 | 是否必须 | 示例 |
|---|---|---|---|
| product_category | 一级品类 | 是 | 洗护 |
| sub_category | 二级子类 | 是 | 修护洗发 |
| efficacy_tags | 功效标签 | 是 | 修护,柔顺 |
| user_tags | 人群标签 | 是 | 学生党 |
| scene_tags | 场景标签 | 是 | 宿舍,开学季 |
| emotion_tags | 情绪标签 | 建议 | 高级感,治愈感 |
| price_band | 价格带 | 建议 | 100-150元 |
| risk_notes | 风险备注 | 建议 | 价格敏感型用户较多 |

## 四、输出字段（模型/人工生成）
| 字段名 | 含义 | 是否必须 | 备注 |
|---|---|---|---|
| heat_score | 热度评分 | 是 | 可自定义加权 |
| potential_score | 传播潜力评分 | 是 | 1-10 |
| target_persona | 高潜人群 | 是 | 示例：学生党 |
| best_time | 最佳推广时间 | 建议 | 示例：开学季 |
| recommendation | 推荐策略 | 建议 | 示例：主推宿舍场景 |
| content_direction | 内容方向 | 建议 | 示例：平价修护+香氛感 |

## 五、建议关键词池（第一轮）
- 修护洗发
- 柔顺洗发
- 学生党洗发水
- 宿舍洗护
- 玫瑰香氛洗发水
- 留香洗发水
- 高性价比洗发水

## 六、最低采集要求
- 第一轮样本量：100 条
- 平台优先级：小红书优先
- 每条数据至少补齐：标题、正文、点赞、评论、发布时间、关键词、标签
